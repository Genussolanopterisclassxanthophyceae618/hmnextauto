## 📖 About this project

This software helps you automate tasks on devices that run HarmonyOS NEXT. You do not need to learn complex code to use it. The program works like other common automation tools. You can record actions or set up scripts to click buttons and navigate screens on your connected mobile device. 

The goal of this tool is to remove manual effort from repetitive mobile tasks. It handles the connection between your computer and the phone. It sends commands to the device screen. You save time by letting the software manage these inputs for you.

## ⚙️ Requirements

Before you start, ensure your computer meets these needs:

- Operating System: Windows 10 or Windows 11.
- Connection: A USB cable to link your device to your PC.
- Device Access: You must enable Developer Options and USB Debugging on your HarmonyOS device.
- Python: This software uses Python to run. The installer includes the necessary components.

## 🚀 Getting started

1. Prepare your HarmonyOS device. Open Settings and go to About Phone. Tap the Build Number seven times. This turns on Developer Options.
2. Go to the Developer Options menu in your settings. Toggle USB Debugging to the on position.
3. Connect your device to your computer with a USB cable. Accept any trust prompts that appear on your phone screen.
4. Visit the link below to get the setup files.

## 🛠️ How to use the software

Once you have installed the program, you can start automating your device. 

1. Launch the hmnextauto application from your desktop or start menu. 
2. The main window will open. Click the Scan button to find your connected phone. 
3. The software will show a status message if it finds your device. If it does not find the device, check your USB connection and verify USB Debugging is active on your phone.
4. Use the interface to select your actions. You can mimic taps, swipes, and text input. 
5. Press the Start button to run your automation. 

The software uses a clear system for defining tasks. You click an element on the screen preview and tell the software what action to perform. You can build a list of these actions to create a sequence.

## 💡 Troubleshooting common issues

If the software fails to connect:

- Change the USB port on your computer. Some ports have better power delivery than others.
- Ensure you use a high-quality data cable. Cheap charging cables often fail to transfer data.
- Check the device trust settings. Sometimes the phone revokes permission to connect to a new computer.
- Restart both the application and the phone. This fixes most communication errors between the two devices.

If the software stops during a task:

- Check your network signal.
- Ensure the phone stays awake.
- Avoid touching the screen while the software performs its automated path. Physical touch can interrupt the command flow.

## 📋 Tips for success

- Keep your scripts short at first. Test one action before building a long chain of commands.
- Use the screen preview to verify the software identifies the buttons you want to click. If the software highlights the wrong area, adjust your settings.
- Rename your scripts after saving them. This makes it easier to track your different automation goals. 
- Watch the log panel during your first few runs. It offers clues if a step fails.

## 🏗️ Managing your workflow

Automation works best when you plan your steps. Break down your manual processes into small parts. For example, if you want to open an app and click a button, identify the exact location of those items in the preview window. The software performs these actions in the order you list them. If you need to wait for a page to load, add a delay step between actions. This ensures the software does not try to click before the button appears. 

Take time to learn the preview tool. It shows the structure of the phone screen clearly. You can click on any menu or button to verify it responds. This saves time during setup. Consistent naming of your files helps keep your workspace clean. You can store these files in your documents folder and move them to a different folder if you reorganize your computer.
